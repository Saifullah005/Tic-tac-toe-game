# Tic Tac Toe in C:
A simple C project of tic tac toe game

## Features: 
-2 player game 
-Basic turns validation
-Winner check and draw logic

## Skills practiced:
-Arrays 
-Structures 
-Conditional logic
-console input output

## How to Run:
1. Compile using any C compiler(GCC)
2. Play via terminal/command line
